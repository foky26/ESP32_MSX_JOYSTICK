Now you can use your old MSX Joystick with you PC for a MSX Emulator.
All you need is a ESP32 Dev Kit with OTG output and a DB9 connector board.
You will need the library https://github.com/schnoog/Joystick_ESP32S2.
Download and install in Arduino IDE.
